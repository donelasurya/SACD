•train – contains training data ( subject_train.txt, X_train.txt, y_train.txt, Inertial Signals )
•test – contains test data ( subject_test.txt, X_test.txt, y_test.txt, Inertial Signals )
•activities_labels.txt - Contains labels for various activities
•features.txt - contains information on various features measured.
